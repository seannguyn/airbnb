# portBnB - An Amazing Accommodation Renting Application
## Front-End UI: React
## Back-End: Django REST framework

Setup guide:
* Run these commands on terminal
```
pip install -r requirements.txt
```

Run server:
```
cd airbnbClone
```
```
python manage.py runserver
```

Run client:
```
cd frontend
```
```
npm start
```
